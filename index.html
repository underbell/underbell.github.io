자기소개

재미있는 개발자가 되고 싶은 저를 소개합니다.

DevOps 지향하며 Web Server는 Apache httpd, Nginx, Webtob를 WAS는 Tomcat, Jeus를 다루어보았으며, Database는 Oracle을 주로 다루었으며 보조적으로 PostgreSql와 Mysql을 다루어보았습니다. Front-End는 Html, Javascript, CSS3, jQuery를 주로 다루며 UI플랫폼은 웹스퀘어5를 다루어 보았습니다. Back-End 언어는 Java를 사용하며 Kotlin에 관심이 있습니다. 웹 Framework는 Spring, Spring Boot를 주로 다루며 웹 보안은 Spring Security를 사용하며 DB 프레임워크로 Mybatis 와 ORM으로 JPA를 주로 다루어보았습니다. WAS 모니터링은 APM으로 Scouter와 JENNIFER 사용해 보았고, 웹서버 로그 분석은 ELK, 버전 관리로 Svn, Git을 사용하며 CI로 Jenkins를 주로 사용하였습니다. 품질관리로 SonarQube를 사용하여 Java 품질검사 도구로 Findbug를 사용하며 이슈관리로 Redmine, Mantis 등을 사용하였습니다.

최근에는 함수형 프로그래밍, Docker, React, Spring-boot, JPA 등을 공부하고 있으며 Github 컨트리뷰션 활동을 늘리며 재미있고 즐거운 개발을 하기 위해 노력 중입니다.

어린 시절에는 사람은 항상 자기가 맡은 일에는 최선을 다해야 하고 결과를 책임져야 한다고 말씀하시는 아버지의 영향을 받은 저는 매사를 신중하게 처리하려 하였고, 정직함과 성실함을 최고의 가치로 여기며 생활해 왔습니다. 항상 자신의 문제는 스스로 해결하고자 노력하였으며 그 결과 독립심과 책임감이 강한 사람으로 성장할 수 있었습니다.
